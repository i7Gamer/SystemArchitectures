Ausführbare Jar-Datei, welche pull und push untersützt und einen Ordner namens "aufgabe2" benötigt,
in welchem das "loetstellen.jpg" Bild enthalten sein muss. Ein LoetstellenFiltered.png wird erstellt.
Ebenfalls wird ein "QualityInfo.txt" erstellt welches die gefundenen Stellen mit den erwarteten Stellen vergleicht.
Diese beide Dateien werden ebenfalls im "aufgabe2" Ordner erstellt.
